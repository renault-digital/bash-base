# Shorten URL

```shell
curl -i https://git.io -F "url=https://raw.githubusercontent.com/renault-digital/bash-base/master/scripts/install.sh" -F "code=JscdA" #  https://git.io/JscdA

curl -i https://git.io -F "url=https://raw.githubusercontent.com/renault-digital/bash-base/master/scripts/install.sh?" -F "code=bash-base/install" #  https://git.io/bash-base/install

curl -i https://git.io -F "url=https://raw.githubusercontent.com/renault-digital/bash-base/master/bin/bash-base.sh" -F "code=bash-base" #  https://git.io/bash-base
```
